class  Order
 

Detailed Description

Die Klasse Order ist ein Objekt zur Speicherung von Bestellungen an einen Node.

Kommt eine Anforderung im Hub an, wird diese zunächst im Orderbuffer gespeichert. Besteht Kontakt zu einem Node (bei ankommendem Heartbeat) werden alle offenen Orderbuffereinträge für diesen Node zu einer Orderstruktur (in der Klasse Order) aggregiert. In einer Order können 6 Orderbuffereinträge zum Node übertragen werden.

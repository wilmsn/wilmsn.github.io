◆ mqtt_stat()

String & Base_Generic::mqtt_stat ( )

Aktuelle Statuswerte werden hier für die MQTT Übertragung zurückgegeben.

Die Befüllung der zugrundeliegenden Variablen "obj_mqtt_state" erfolgt in dem abgeleiteten Script.

Der Eintrag erfolgt entweder als Einzelwert oder als JSON formatierter String.

Eine Übertragung per MQTT erfolgt nach jeder Änderung getriggert durch mqtt_stat_changed()

Ein String

Systeminformationen zum Sensor für die Webseite als json abgespeichert;.

Dieser String muss durch das abgeleitete Objekt gefüllt werden. Dabei gilt für jeden Systeminfowert:

""obj_html_placeX"+":"+"obj_labelX"+"MesswertX"+"EinheitX", ... "

Hier muss immer ein komplettes, gültiges Teil-JSON stehen dazu wird als default ein Dummy eingetragen.

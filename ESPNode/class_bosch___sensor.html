Detailed Description

Ein abgeleitetes Objekt für einen Bosch Sensor. Benötigt: https://github.com/wilmsn/BMX_sensor als Gerätetreiber Das Objekt kann entweder für 2 Meßwerte (Temperatur und Luftdruck) BMP180/BMP280 oder für 3 Meßwerte (+ Luftfeuchte) BMP280 initialisiert werden.

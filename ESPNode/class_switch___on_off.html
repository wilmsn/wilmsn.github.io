Detailed Description

Ein abgeleitetes Objekt für einen Ein-Aus Schalter.
Der Schalter reagiert nur wenn es in der set Funktion mit dem Schlüsselwort aufgerufen wurde das bei der Initialisierung hinterlegt wurde.
Folgende Strings als value übergeben schalten aus:
0 aus Aus off Off
Folgende Strings als value übergeben schalten ein:
1 ein Ein on On
Folgende Strings als value übergeben schalten um:
2 umschalten Umschalten toggle Toggle

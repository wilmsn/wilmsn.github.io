Detailed Description

Ein generisches Objekt für einen Sensor. Nicht zum Einbau bestimmt, sondern nur als Vererbungsobjekt. Hier werden Grundfähigkeiten für jeden Sensor oder Switch erstellt. Funktionsfähig sind: Die "begin" Funktion zum Einbau ins "setup" des Hauptprogrammes. Diese ist ggf. in einem abgeleiteten Objekt um zusätzliche Parameter zu erweitern. "set_label" Funktion zum nachträglichen setzen oder verändern eines Labels. "set_keyword" Funktion zum nachträglichen setzen oder verändern eines Keywords. "set_mqtt_name" Funktion um den Namen für die Übertragung im JSON String festzulegen. "set_hw_pin" Funktion um bis zu 2 Hardwareanschlüsse festzulegen. "changed" Funktion um eine Statusveränderung zu erkennen "keyword_match" Funktion um zu prüfen ob der Sensor durch das übergebene Keyword angesprochen wird. "*json" Funktionen um (Teil)Stücke eines JSON Statements zu erzeugen. "show*" Funktionen um gespeicherte Inhalte auszugeben. Als folgende Funktionen sind aus Gründen der Kompatibilität leer (als Dummy) implementiert: "loop" Zum Aufruf in der Hauptschleife des Hauptprogrammes. Hier können in einem abgeleitetem Objekt regelmäßige Tätigkeiten implementiert werden. "web_create_json" Hier wird durch das abgeleitete Objekt das JSON zur Erzeugung des Sensors/Actors auf der Webseite erstellt.
